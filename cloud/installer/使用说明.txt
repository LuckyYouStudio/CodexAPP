CodexApp Agent —— 让手机/网页远程控制这台电脑上的 Codex

【最简单：直接运行（推荐先这样测试）】
1. 确保这台电脑已安装并登录 Codex（桌面端能正常用即可）。
2. 双击 CodexApp-Agent.exe。
3. 会自动弹出浏览器，打开本机控制面板 http://127.0.0.1:7878
4. 在面板里填 Broker 地址、邮箱、密码 → 点「登录并启动」。
   （没有账号就点「注册」标签注册，按邮件验证后再登录。）
5. 登录后面板会显示状态：Broker 已连接 / 本地 Codex 已就绪 / 配对码。
6. 在手机 App 或网页客户端用同一账号登录，首次连接输入面板上的「配对码」即可。
   之后就能发提示词、审批命令、看代码改动(diff)，远程控制这台电脑的 Codex。

  面板会实时显示连到哪一步了（登录中/连接中/等待客户端/已连接），
  不会再「盲转圈」。随时可在浏览器打开 http://127.0.0.1:7878 查看。

【正式安装：开机自启、后台无窗口运行】
1. 把 CodexApp-Agent.exe 和本文件夹里的脚本放在一起。
2. 双击 Install.cmd，按提示填 Broker 地址、邮箱、密码、默认项目目录。
3. 之后 Agent 会后台运行并开机自启（无窗口，不再弹浏览器）。
   需要时仍可手动打开 http://127.0.0.1:7878 查看状态 / 重新登录。

【改配置 / 换账号】
最简单：打开 http://127.0.0.1:7878 →「退出登录」后重新登录即可。
或编辑 agent.config.json（和 exe 同目录；安装版在 %LOCALAPPDATA%\CodexAppAgent\）。

【卸载】
双击 Uninstall.cmd。

【注意】
- 电脑要保持开机、不休眠，客户端才连得上（Codex 在本地运行）。
- 所有内容端到端加密，服务器看不到你的代码和命令。
- 控制面板只监听本机 127.0.0.1，外部访问不到。
- 本程序未做代码签名，Windows 可能弹 SmartScreen 提示，点“更多信息→仍要运行”。
  （正式发布版会做代码签名，不再有此提示。）
